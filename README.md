# DTS101-Data-Science-Tools-practical-I
DTS101 Data Science Tools practical I

Data visualization and Storytelling (Practical)
Loading and linking datasets: cleaning data and creating calculated columns and measures, reports, data and relationship views; 
Numeric and graphic visualizations: cards, tables, line chart, bar chart, pie chart, column chart, treemap, using slicers and custom visuals; 
Planning designing and prototyping: working with various charts, working with filters; visual storytelling: telling stories with visuals, when to use which Visuals, presentation best practices; 
Functions and control flows: creating and working with functions, conditional statements, for and while loops; 
Data structures: lists, turples, sets; working with data frames, plots and graphs; 
Exploratory data analysis: statistical measures, probabilities and hypotheses, algorithms and algorithmic complexity, advanced interactive visual analysis
